Questão 1

O que é Internet e WEB? São as mesmas coisas? Explique de forma clara e evidencie as diferenças.

A Internet é uma rede global que conecta os computadores, compartilhando informações. Já a WEB ém serviço que roda sobre a rede mundial de computadores. Ou seja elas não são a mesma coisa, mas sim, a WEB é algo que faz parte da Internet

Questão 2

O que é arquitetura cliente-servidor utilizada na web? Explique o carregamento de uma página passo a passo.

Essa arquitetura um computador é o cliente e outro computador é o servidor, esse primeiro realizando um pedido (uma pesquisa) e o segundo fornecendo a resposta.
Ao realizar o carregamento de uma página o computador que está sendo usado trabalha como o cliente enviando essa pesquisa (pedido) para o servidor, que irá encaminhar a resposta de volta para o computador que está pesquisando.

Questão 3

O que é um tag HTML?

Um tag é um marcador que permite reconhecer qual elemento gráfico que será utilizado, definindo o conteúdo da página web.

Questão 4

O que é HTML e qual a estrutura básica de um documento HTML?

A linguagem HTML, ou HyperTextLanguage, é usada para o desenvolvimento de páginas web, utilizando tags para estruturar as páginas.
Sua estrutura basicamente é dividida em um Head, no qual fica o título do site, e um body, no qual localiza-se a estrutura da página, podendo ser composta por diversos elementos, como títulos, links, fotos, estre outros. Além disso normalmente ao criar-se um documento HTML, deve-se colocar em seu inicio o elemento DOCTYPE html para definir o documento como HTML, quando ele for lido
