Questão1

O que é Internet e WEB? São as mesmas coisas? Explique de forma clara e evidencie as diferenças.

A internet é uma rede global que conecta computadores, compartilhando informações. Já a WEB é um serviço que roda sobre a rede mundial de computadores. Ou seja, elas não são a mesma coisa, mas sim, a WEB faz parte da Internet

Questão 2

O que é arquitetura cliente-servidor utilizada na web? Explique o carregamento de uma página passo a passo.

Nessa arquitetura, um computador é o cliente e outro é o servidor, esse primeiro realiza um pedido (uma pesquisa) e o segundo fornece a resposta.
Ao realizar o carregamento de uma página, o computador, que está sendo usado, trabalha como o cliente enviando essa requisição para o servidor, que irá encaminhar a resposta de volta ao computador que está pesquisando.

Questão 3

O que é um tag HTML?

Uma tag é um marcador que permite reconhecer qual elemento gráfico será utilizado, definindo o conteúdo da página web.

Questão 4

O que é HTML e qual a estrutura básica de um documento HTML?

A linguagem HTML é usada para o desenvolviemnto de páginas web, utilizando tags para estruturar as páginas.
Sua estrutura basicamente é dividida em um Head, no qual fica o titulo do site, e um body no qual localiza-se a estrutura da página, podendo ser composta por diversos elementos, como títulos, links, fotos, entre outros. Além disse, ao criar o arquivo HTML, recomenda-se colocar o elemento DOCTYPE html para definir o documento como HTML, quando ele for lido/compilado.

Questão 5

Discuta com colega o que são tags semânticas do HTML.

As tags semânticas são utilizadas para especificar o que cada parte do site é, dando um "contexto" para determinada parte do código, ajudando leitores a entender a estrura do código. Um exemplo é "header".
